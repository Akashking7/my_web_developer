1. DROP TABLE IF EXISTS
DROP TABLE IF EXISTS visited_countries, users;
Deletes tables only if they exist
Prevents errors
Same as deleting a table in pgAdmin UI

👉 Used for resetting database

🟦 2. UPDATE (Modify existing data)
UPDATE users
SET name = 'Angelina'
WHERE id = 1;
Changes data in a table
SET → new value
WHERE → selects specific row

👉 Without WHERE, all rows will be updated ⚠️

🟦 3. ORDER BY (Sort data)
Ascending (default)
SELECT * FROM users
ORDER BY id ASC;
Descending
SELECT * FROM users
ORDER BY name DESC;
ASC → A → Z / small → large
DESC → Z → A / large → small
🟦 4. DELETE (Remove data, not table)
DELETE FROM visited_countries
WHERE id = 6;
Deletes specific row
Table structure remains
Multiple condition delete
DELETE FROM visited_countries
WHERE user_id = 1 AND country_code = 'FR';
Deletes rows matching both conditions
🔴 Important Difference
DROP TABLE → deletes entire table
DELETE → deletes only data (rows)
